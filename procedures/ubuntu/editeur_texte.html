Éditeur de texte

Il n'est pas possible de créer des fichiers textes comme dans Windows en utilisant le menu contextuel dans l'Explorateur de fichiers.

Vous devez ouvrir l'utilitaire Éditeur de texte dans le menu.

1. Créer le fichier​

  1. Dans le menu, sélectionnez l'application Éditeur de texte.

  1. Inscrivez le texte dans l'éditeur.

  2. Appuyez sur le bouton Enregistrer.

  3. Sélectionnez le dossier de destination et inscrivez le nom du fichier. Appuyez sur le bouton Enregistrer pour finaliser.

2. Mode administrateur​

Pour certains dossiers ou partitions, il faut être administrateur pour être en mesure de créer des fichiers.

  1. Dans le terminal, tapez la commande sudo gedit.

  2. La fenêtre qui s'ouvre sera en mode administrateur. Il sera possible d'enregistrer dans le dossier protégé.

    Important

    Il ne faut pas fermer la fenêtre du terminal, il s'agit du processus parent de l'Éditeur de texte en mode administrateur.

    Si vous fermez le terminal, l'Explorateur en mode administrateur se fermera également.
