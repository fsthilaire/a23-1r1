Utiliser Ubuntu Live

Il est possible d'utiliser Ubuntu sans aucune installation. Il s'agit du mode Live.

Ceci peut être pratique pour effectuer des actions qui ne sont pas possibles de faire facilement en Windows.

  1. Au démarrage du programme, il faut choisir le mode d'installation Sélectionnez Try or Install Ubuntu. Appuyez sur Enter pour confirmer.

  1. Dans la section de gauche, sélectionnez la langue à installer.

  2. Appuyez sur le bouton Essayer Ubuntu.

  3. La disposition du clavier par défaut est AZERTY. Pour la modifier, appuyez sur fr1 en haut à droite.

  4. Sélectionnez Français (Canada) dans la liste.
