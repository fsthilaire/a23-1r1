Installation de VirtualBox

Prérequis BIOS​

Pour être en mesure d'utiliser VirtualBox, la virtualisation doit être activée dans le BIOS.

Généralement, cette option est activé par défaut.

Prérequis Visual C++ redistributable​

Vous devez installer les fichiers C++ redistributable. Il est possible que vous l'ayez déjà dans votre ordinateur.

Pour télécharger : https://aka.ms/vs/17/release/vc_redist.x64.exe

Il suffit d'exécuter le programme vc_redist.x64.exe.

Acceptez les conditions et appuyez sur le bouton Installer.

Téléchargement​

Téléchargez le programme d'installation de VirtualBox 7.0.10.

Installation​

  1. Exécutez le programme d'installation VirtualBox-7.0.10-158379-Win.exe.

  2. Appuyez sur le bouton Suivant >.

  3. Appuyez sur le bouton Suivant > pour conserver les fonctionnalités par défaut.

  4. Le logiciel indique que les cartes réseau devront être interrompues. Si vous avez des téléchargements en cours, ils seront arrêtés. Appuyez sur le bouton Oui.

  5. Il faut accepter d'installer les librairies manquantes pour que le logiciel fonctionne. Appuyez sur le bouton Oui.

  6. Appuyez sur le bouton Installer pour débuter l'installation.

  7. Appuyez sur le bouton Terminer pour conclure l'installation.
