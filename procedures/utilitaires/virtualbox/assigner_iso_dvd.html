Assigner un fichier ISO au lecteur DVD

  1. Sélectionnez la machine virtuelle à modifier.

  2. Appuyez sur le bouton Configuration.

  3. Sélectionnez l'item Stockage dans le menu de gauche.

  4. Sélectionnez le lecteur DVD dans la section Unités de stockage.

  5. Appuyez sur le bouton Choisissez un lecteur optique... dans la section Attributs.

  6. Sélectionnez l'item Choose a disk file....

  7. Dans la fenêtre de l'Explorateur de fichiers, sélectionnez le fichier ISO. Appuyez sur le bouton Ouvrir.

  8. Le fichier ISO sera indiqué dans l'emplacement du lecteur DVD de la section Unités de stockage. Appuyez sur le bouton OK
