Ventoy

Ventoy est un logiciel qui permet de créer une unité de stockage démarrable avec plusieurs images.

La version présentée est la 1.0.95.

Pour plus d'information, veuillez visiter le site de Ventoy.

Téléchargement​

Téléchargez la version Windows de Ventoy.

Installation​

Il s'agit d'un logiciel portable.

Veuillez extraire le fichier zip dans un dossier.

Je recommande le dossier c:\outils\ventoy.

Créer une clé démarrable​

Insérez la clé USB dans l'ordinateur.

Démarrez le programme Ventoy2Disk.exe en tant qu'administrateur.

Sélectionnez la clé USB dans la liste déroulante.

Dans le menu Option, décochez l'item Secure Boot Support.

Dans le menu Option -> Partition Style, cochez GPT.

Appuyez sur le bouton Install.

remarque

Si la clé contient déjà Ventoy, il y aura une fenêtre de confirmation additionnelle.

Il faut inscrire YES dans la zone de texte.

Appuyez sur le bouton Oui pour confirmer l'avertissement que le contenu de la clé sera effacé.

Appuyez sur le bouton Oui pour confirmer le 2e avertissement que le contenu de la clé sera effacé.

Appuyez sur le bouton OK lorsque la création est terminée.

La partie de droite indique la version de Ventoy de la clé sélectionnée.

Copie du ISO​

Trouvez la lettre du lecteur de la partition Ventoy de la clé USB.

Directement à la racine de la clé, il faut copier les fichiers ISO.

danger

Si vous voyez une partition nommée VTOYEFI en FAT de 32 Mo, il est important de ne pas l'utiliser.

Utilisation​

Vous devez démarrer l'ordinateur à partir de sur la clé USB.

Sur les ordinateurs Dell, la touche est F12 permet de choisir l'unité de stockage pour le démarrage lors du démarrage de l'ordinateur.

Avertissement

Le nom de la clé USB sera différent en fonction du modèle utilisé.

L'apparence du menu sera différente en fonction de l'ordinateur.

Lorsque la clé est chargée en mémoire, il faut sélectionner le fichier ISO à utiliser dans la liste avec le clavier.

Ensuite, il faut choisir l'item Boot in normal mode.
