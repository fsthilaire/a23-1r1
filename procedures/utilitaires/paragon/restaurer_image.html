Restaurer une image

Important

Avant de restaurer une image, il faut s'assurer que la taille du disque qui recevra l'image est égale et supérieure au disque original.

Il est recommandé d'effacer toutes les partitions du disque avant de débuter.

  1. Sélectionnez l'item Restore from file.

  2. Sélectionnez l'item Restore volumes dans le menu de gauche.

  3. Dans l'Explorateur de fichiers, sélectionnez le sous-dossier Job- qui se retrouve dans le dossier de l'image.

  4. Appuyez sur le bouton Next >.

  5. Le logiciel sélectionne automatiquement le disque vierge de l'ordinateur. Il est important de vérifier que le disque est le bon. Vérifiez le modèle et la taille du disque.

  6. Appuyez sur le bouton Restore now.

  1. Appuyez sur le bouton Yes pour confirmer la restauration. Le message indique que le contenu du disque sera complètement effacé par le contenu de l'image.

  2. Le processus de restauration peut prendre plusieurs minutes. Appuyez sur le bouton Close lorsque la restauration est terminée.

  3. Pour fermer l'ordinateur, appuyez sur le bouton en bas à gauche et sélectionnez l'item Shut down.
