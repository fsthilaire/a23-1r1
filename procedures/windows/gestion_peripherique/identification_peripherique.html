Identification d'un périphérique

S'il n'est pas possible de trouver de l'information sur la pièce, il est possible de trouver son identifiant système.

  1. Effectuez un clic-droit sur le périphérique à identifier et sélectionnez Propriétés.

  1. Dans la fenêtre, sélectionnez l'onglet Détails.

  1. Dans la liste déroulante Propriété, sélectionnez Numéro d'identification du matériel.

  1. Prenez la première ligne.
  • En jaune, c'est le type de périphérique (USB, PCI)
  • En vert, c'est le numéro du fournisseur (VID_)
  • En bleu c'est le numéro de la pièce (PID_)

  1. Allez sur le site https://devicehunt.com/ pour trouver le nom officiel du périphérique.

  1. Entrez l'information obtenue à étape 4.

    Dans la section Type (jaune), entrez le type obtenu (PCI ou USB)

    Dans la section Vendor ID (vert), entrez le numéro du fournisseur. Ne pas inscrire VID_.

    Dans la section Device ID (bleu), entrez le numéro de la pièce. Ne pas inscrire PID_.

  2. Appuyez sur le bouton Rechercher pour trouver l'information détaillée de la pièce.

  3. Il est possible d'obtenir le nom du fournisseur et le nom du produit.

  4. Il reste à rechercher sur le site du fournisseur les pilotes en fonction du numéro de la pièce.

    Important

    Ne pas télécharger le pilote à partir du site DeviceHunt si c'est proposé.

    Il est toujours recommandé de télécharger le pilote directement du site du fournisseur.
