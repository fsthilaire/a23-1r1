Accès au gestionnaire d'applications

Le gestionnaire d'applications est l'endroit où Windows regroupe l'information de tous les logiciels installés.

C'est à cet endroit qu'il est possible de gérer les désinstallations et d'assigner les programmes par défaut en fonction de l'extension du fichier.

Voici la procédure pour accéder au gestionnaire d'applications.

Windows 10​

Appuyez sur le menu démarrer et sélectionnez l'item Paramètres.

Dans la fenêtre des Paramètres de Windows, sélectionnez l'item Applis.

Windows 11​

Appuyez sur le menu démarrer et sélectionnez l'item Paramètres.

Dans la fenêtre des Paramètres de Windows, sélectionnez l'item Applis.
