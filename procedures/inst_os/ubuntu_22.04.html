Ubuntu 22.04

Voici la procédure d'installation.

Des les éléments théoriques seront intégrés à chacune des sections.

Veuillez démarrer le programme d'installation de Ubuntu à partir du ISO.

Il est préférable d'activer uniquement le disque interne pour l'installation dans le BIOS. Ceci facilitera la création des partitions.

1. Démarrage​

  1. Au démarrage du programme, il faut choisir le mode d'installation Sélectionnez Try or Install Ubuntu. Appuyez sur Enter pour confirmer.

  1. Dans la section de gauche, sélectionnez la langue à installer.

  2. Appuyez sur le bouton Installer Ubuntu.

  3. Sélectionnez la disposition du clavier et appuyez sur le bouton Continuer.

  4. Sélectionnez les options d'installation. Par défaut, sélectionnez uniquement Installation minimale. Appuyez sur le bouton Continuer.

    remarque

    L'option minimale installe uniquement le système d'exploitation avec les utilitaires de base.

    L'option installation normale installe plusieurs utilitaires pratiques comme GParted et Libre Office.

    L'option Télécharger les mises à jour... permet d'installer les dernières mises à jour disponible en même temps que l'installation. L'installation initiale sera plus longue si cette option est sélectionnée, mais il est recommandé de faire les mises à jour le plus rapidement possible.

  5. Sélectionnez la ville la plus proche de votre secteur. Pour le Québec, la ville de Toronto est suggérée. Appuyez sur le bouton Continuer.

2. Création des partitions​

  1. Sélectionnez l'option Effacer le disque et installer Ubuntu. Il est préférable de laisser le programme d'installation de créer les partitions.

  2. Appuyez sur le bouton Continuer pour accepter la création des partitions.

3. Configuration​

  1. Dans le champ Votre nom, veuillez inscrire le nom de l'utilisateur principal.

  2. Dans le champ Le nom de votre ordinateur, veuillez inscrire le nom de l'ordinateur.

  3. Dans le champ Choisir un nom d'utilisateur, veuillez inscrire le nom de l'utilisateur principal de l'ordinateur.

  4. Inscrivez le mot de passe de l'utilisateur dans les champs Choisir un mot de passe et Confirmez votre mot de passe.

  5. Appuyez sur le bouton Continuer pour accepter la configuration.

4. Finalisation​

  1. Appuyez sur le bouton Redémarrer maintenant pour finaliser l'installation.

  2. Retirez la clé USB et appuyez sur la touche ENTER pour redémarrer l'ordinateur.

  3. Connectez-vous avec votre mot de passe.

  4. Appuyez sur le bouton Passer.

  5. Sélectionnez l'option Ignorer pour le moment pour le service Ubuntu Pro. Appuyez sur le bouton Suivant.

  6. Sélectionnez l'option Non, ne pas envoyer d'informations système et appuyez sur le bouton Suivant.

  7. N'activez pas le service de géolocalisation. Appuyez sur le bouton Suivant.

  8. Appuyez sur le bouton Terminer pour finaliser la personnalisation.
