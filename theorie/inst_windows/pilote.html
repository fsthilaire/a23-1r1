Pilote (driver)

Un pilote dans un système d'exploitation est un programme qui indique au système d’exploitation comment communiquer adéquatement avec un périphérique. En anglais c'est un driver.

Windows 10 et Windows 11 contiennent une banque de pilotes génériques et des pilotes spécifiques d'une très grande liste de pièces. L'ordinateur peut être fonctionnel à 100% dès que l'installation de Windows est terminée.

Il arrive parfois que des pièces ne soient pas reconnues dans Windows automatiquement. Il faut donc installer manuellement les pilotes.

Les fabricants de périphériques et d'ordinateurs distribuent généralement leurs pilotes sur leur site web.

Il faut trouver le bon pilote en fonction du logiciel. Il n'est pas recommandé d'utiliser un pilote d'une source non vérifiée, car le pilote ne sera peut-être pas compatible ou aura été modifié.

Il arrive parfois lorsque la pièce n'est plus supportée par le fournisseur que le pilote ne soit plus disponible. Il est important de conserver les pilotes des pièces.

Il est important de prendre le pilote en fonction de l'architecture de l'ordinateur (x32 ou x64), si l'option existe.

1. Identification des périphériques sans pilote​

L'utilitaire Gestionnaire des périphériques est l'endroit qui permet de gérer les périphériques de Windows.

Pour y accéder rapidement, il faut utiliser le menu Démarrer avancé (clic droit sur le menu Démarrer).

Les périphériques avec des points d'exclamation ne fonctionneront pas ou seront limités.

Dans l'image ci-dessus, il y a 4 périphériques dans la section Autres périphériques. Ces périphériques ne seront pas fonctionnels.

Dans la section Cartes graphiques, le pilote de la carte graphique est un pilote générique. Il sera très limité dans son fonctionnement. Généralement, c'est uniquement le port VGA qui est fonctionnel. Il faudra la mettre à jour.

Dans la section Contrôleurs audio, vidéo et jeu, il y a un des deux périphériques avec un mauvais pilote.

Il est important que tous les périphériques soient correctement installés avec le bon pilote.

Les noms ne sont pas très significatifs pour nous aider à trouver la bonne pièce. Il faut regarder dans les propriétés pour avoir le nom système ou de regarder sur la pièce les informations pertinentes.

2. Fichier inf​

Windows utilise l’extension .inf pour contenir l’information de l’installation du pilote dans l’ordinateur.

Ce fichier est utilisé lors d'une installation par le gestionnaire de périphériques.

3. Exécutable​

Les pilotes peuvent s’installer à partir d’un fichier exécutable également.

Le programme s'occupe d'installer les fichiers dans l'ordinateur et les associes au bon pilote.

4. Outil de détection - Dell Support Assist​

Les fabricants d'ordinateurs, comme Dell, offrent généralement des outils de détection de pilotes.

Celui de Dell est Support Assist.

Il est important d'installer ce logiciel sur l'ordinateur qui doit être configuré et non sur notre propre ordinateur.

Le logiciel fait une détection du matériel et télécharge les pilotes manquant et non à jour.

Pour pouvoir utiliser cet outil, il faut minimalement avoir une connexion à Internet. Si la carte réseau n'est pas disponible, il faut installer manuellement le pilote de la carte réseau.

Pour utiliser le logiciel, il faut aller dans la fiche de l'ordinateur à partir du Service Tag. Les pilotes et l'accès à l'outil se retrouvent dans la section Pilotes et téléchargements.

Important

Le fonctionnement de l'outil sera fait en laboratoire.

Il n'y aura pas de procédure dans les notes de cours.

Il sera important de prendre des notes.

5. Windows Update​

Windows Update est en mesure de détecter également certains pilotes et de les installer automatiquement.

6. Compatibilité des pilotes​

Il arrive parfois que la pièce ne possède pas de pilotes pour une version de Windows.

Il est possible que le pilote d'une autre version de Windows fonctionne correctement.

Les pilotes de Windows 10 fonctionnent pratiquement toujours pour Windows 11.

Les pilotes de Windows 7 et 8 fonctionnent généralement pour Windows 10 et Windows 11.

Les pilotes de Vista et XP fonctionnent très rarement pour Windows 10 et Windows 11.
