Concepts de base

Voici quelques concepts de base à prendre en compte lors de l'installation de Windows.

1. Architecture processeur​

Pour être en mesure d'installer un système d'exploitation, il faut prendre une version qui est compatible avec le processeur.

Pour les systèmes d'exploitation, les 4 architectures principales sont :

  • Intel 32 bit
  • Intel 64 bit
  • ARM 32 bit
  • ARM 64 bit

Pour Windows 11, il y a seulement la version Intel 64 bit qui est disponible. Il n'existe plus de version Intel 32 bit.

remarque

Il existe une version ARM 64 bit de Windows, mais elle n'est pas disponible au grand public.

La dernière version de Windows Intel 32 bit est Windows 10 2004.

De nos jours, les processeurs des ordinateurs de type PC sont tous 64 bit.

Pour les systèmes d'exploitation plus anciens, il est possible d'installer une version Intel 32 bit sur un processeur Intel 64 bit. Par contre, il ne sera pas possible d'utiliser des logiciels x64 et la mémoire vive sera limitée à 4 Gio.

2. Obtenir le fichier ISO​

Pour obtenir le ISO officiel, il faut aller sur le site : https://www.microsoft.com/fr-ca/software-download/windows11

Par la suite, il faut créer une clé démarrable avec Ventoy ou Rufus.

3. Licence​

Les systèmes d'exploitation Windows ne sont pas gratuits.

Les ordinateurs de fabricants ont généralement une licence de Windows incluse avec l'ordinateur. Cette licence s'appelle une licence OEM. Une licence OEM ne peut pas être transférée sur un autre ordinateur. Une partie du coût d'achat de l'ordinateur consiste à l'achat de Windows.

Pour être en mesure de transférer une licence entre 2 ordinateurs, il faut une licence Retail. Ce type de licence permet uniquement d'avoir une installation active. Il faut que l'ancien ordinateur n'utilise plus Windows.

Pour les entreprises qui possèdent un très grand parc informatique, une licence de type Volume Licencing ou VL sont généralement utilisées.

Une licence OEM est moins dispendieuse, car elle n'est pas transférable.

4. Clé d'installation​

La clé d'installation est nécessaire pour confirmer que l'utilisateur possède une licence valide.

Depuis Windows 8, la clé de Windows se retrouve enregistrée dans le BIOS pour les licences OEM. Pour les ordinateurs plus anciens, la clé se retrouve normalement sur un autocollant Microsoft sur le boitier.

Le programme d'installation la détecte automatiquement. Si la clé est pour la même édition de Windows, elle ne sera pas demandée lors de l'installation.

Si aucune clé n'est détectée lors de l'installation ou l'édition en cours d'installation n'est pas la même que la licence OEM, il faut entrer manuellement la clé.

Pour Windows 10, il est possible d'utiliser une clé d'installation de Windows 7 et de Windows 8. La mise à niveau du système d'exploitation est gratuite.

Pour Windows 12, il est possible d'utiliser une clé d'installation de Windows 7, de Windows 8 et Windows 10. La mise à niveau du système d'exploitation est gratuite.

5. Activation​

Lorsque Windows est installé avec une clé d'installation, il faut que l'installation soit activée.

Dès que Windows se connecte à Internet, Windows communiquera avec son serveur d'activation pour déterminer si la clé est valide.

Selon la version de Windows, l'utilisateur a un délai pour activer son installation de Windows. Après ce délai, Windows limitera certaines fonctionnalités.

Pour plus d'information : https://support.microsoft.com/fr-fr/windows/activer-windows-c39005d4-95ee-b91e-b399-2820fda32227#WindowsVersion=Windows_11

6. Compte utilisateur - nom d'utilisateur​

Un compte utilisateur est ce qui permet de s'identifier à un ordinateur. Il est composer d'un non d'utilisateur et d'un mot de passe.

Le compte permet de donner accès uniquement aux éléments que l'utilisateur peut accéder. Il permet de gérer la sécurité.

Windows permet 3 types de comptes utilisateurs.

  • Compte local

    C'est Windows qui enregistre le mot de passe de l'utilisateur dans sa base de données interne.

  • Compte en ligne

    C'est les serveurs de Microsoft qui gèrent l'authentification.

    L'avantage d'un compte en ligne est qu'il est possible de synchroniser facilement un environnement utilisateur entre plusieurs ordinateurs.

  • Compte d'entreprise

    C'est un serveur dans l'entreprise qui gère l'authentification.

Dans ce cours, ce sera une installation avec un compte local.

Windows 11 force les utilisateurs à utiliser un compte en ligne. Il est possible d'indiquer à Windows lors de l'installation d'ignorer ceci.

7. Nom de l'ordinateur - hostname​

Il ne faut pas confondre le nom de l'ordinateur avec le nom de l'utilisateur.

Le nom de l'ordinateur consiste à donner un nom significatif à l'installation de Windows. Ce nom représente l'ordinateur.

Le nom de l'ordinateur sera visible sur le réseau pour accéder à ses ressources partagées ou pour l'administrer.

Il est important que le nom soit unique lorsque l'ordinateur est dans un réseau d'entreprise.

Lors de l'installation de Windows, un nom d'ordinateur est autogénéré.

8. Vérification matérielle​

Pour Windows 11, si le matériel ne répond pas aux critères de Windows 11, il ne sera pas possible de l'installer.

Avec Ventoy, la vérification matérielle est ignorée automatiquement.

Par contre, il est possible de configurer le programme d'installation pour ignorer les éléments ci-dessous en ajoutant des clés dans la base de registre si l'installation ne se fait pas avec Ventoy et que l'ordinateur n'est pas officiellement compatible.

  • TPM
  • Secure Boot
  • Mémoire vive

Voici les clés à ajouter.

indows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SYSTEM\Setup\LabConfig]
"BypassTPMCheck"=dword:00000001
"BypassSecureBootCheck"=dword:00000001
"BypassRAMCheck"=dword:00000001

Pour avoir la procédure complète, veuillez vous référez au site de lecrabeinfo.
