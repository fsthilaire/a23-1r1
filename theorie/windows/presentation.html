Présentation de Windows

Windows est le système d'exploitation qui est produit par Microsoft.

Il s'agit du système d'exploitation le plus connu, car il est souvent le système préinstallé dans un nouvel ordinateur. Malgré qu'il est souvent inclus dans un nouvel ordinateur, Windows est un système d'exploitation payant. Il portion du prix de l'ordinateur consiste à payer la licence à Microsoft

Windows est un système d'exploitation propriétaire, c'est-à-dire que Microsoft conserve le code et il est entièrement programmé et maintenu par lui.

Windows existe depuis 1985. Par contre, c'est à partir de 1990 que Windows devient plus populaire avec Windows 3.0.

Dans ce cours, Windows 10 et de Windows 11 seront présentées.

Windows 10 est disponible depuis 2015. La version 22H2 (19045.3324) est la dernière version de Windows 10.

Microsoft supportera cette version jusqu'au 14 octobre 2025. Après cette date, il n'y aura plus de mise à jour du système d'exploitation. Les failles de sécurités qui seront découvertes après cette date ne seront plus corrigées.

Cette version est encore largement utilisée en entreprise. Par contre, elle n'est plus vendue pour les nouveaux ordinateurs personnels.

Windows 11 est disponible depuis novembre 2021. La version 22H2 (22621.2134) est la dernière version de Windows 11. Le système a le même identifiant de version que Windows 10, mais le build est différent.

Pour les ordinateurs personnels, Windows 11 est généralement inclus actuellement.

En entreprise, la transition se fait plus tranquillement, mais prendra de la vitesse prochainement, car Windows 10 est en fin de vie.

Édition​

La version et l'édition de Windows sont 2 choses distinctes.

Il existe principalement 2 éditions.

  • Famille ou Home en anglais
    • Cette édition est dédiée principalement pour le grand public et pour un usage domestique.
    • Les fonctionnalités de gestion des comptes et des groupes sont limitées.
    • L’utilitaire pour la gestion des comptes et des groupes est simpliste et peu convivial.
    • Il n’est pas possible d’intégrer l’ordinateur dans un réseau d’entreprise (domaine).
  • Professionnel ou Professionnal en anglais
    • Cette édition est dédiée pour les entreprises.
    • La gestion des comptes et des groupes utilisateurs locaux offre plus d’options.
    • L’utilitaire pour la gestion des comptes et des groupes est similaire à celui de la version serveur.
    • Des fonctionnalités additionnelles sont disponibles pour intégrer un réseau d’entreprise (domaine).

Il existe d’autres éditions, mais elles sont moins populaires :

  • Entreprise;
  • Éducation;
  • Professionnel pour station de travail.

Selon l'édition de Windows, les fonctionnalités et limitations sont différentes.

Version et build​

Il existe plusieurs versions pour un système d'exploitation. Depuis Windows 10, Microsoft publiait 2 versions par année.

Avec l'arrivée de Windows 11, il y a eu seulement une nouvelle version de Windows 10 et de Windows 11 pour l'année 2022.

Chaque version apporte des différences dans Windows. Elles peuvent être mineures et majeures.

Au début, l'identifiant de la version était une série de 4 chiffres. Les 2 premiers chiffres consistaient à l'année de la publication et les 2 derniers au mois de la publication. Donc la version 1909 de Windows 10 a été publiée en septembre 2019. La version 2004 a été la dernière version à utiliser cette nomenclature.

Par la suite, Microsoft utilise H1 pour la première partie de l'année et H2 pour la 2e partie de l'année pour spécifier le mois afin d'être plus uniforme dans leur nomenclature de version. La première version à utiliser cette nomenclature est 20H2.

La version a également un numéro de build. La version 22H2 de Windows 10 correspond à 10.0.19045.3324. Le build ne change pas en fonction de l'édition.

La dernière version de Windows 10 est la 22H2. Elle est disponible depuis le 8 octobre 2022.

La dernière version de Windows 11 est la 22H2. Elle est disponible depuis le 8 octobre 2022.

Pour connaitre la version de Windows à partir du numéro de build, il suffit de consulter ces sites.

  • Windows 10 : https://learn.microsoft.com/fr-ca/windows/release-health/release-information
  • Windows 11 : https://learn.microsoft.com/fr-fr/windows/release-health/windows11-release-information
