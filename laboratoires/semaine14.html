Préparation examen #5 laboratoire

1. Consignes​

L'examen se fera individuellement.

Vous aurez comme ordinateur le Dell Optiplex 9020, Dell Precision 3420 ou Dell Optiplex 7080.

1.1. Clé USB​

Pour être en mesure de faire l'examen, vous devez avoir votre clé USB que vous avez préparée pour le travail #4.

L'enseignant vérifiera lors de ce laboratoire que la clé est correcte.

Important

Si vous n'avez pas votre clé USB, il ne sera pas possible de faire l'examen.

1.3. Ressources autorisées​

Vous avez accès à toute la documentation.

  • Notes de cours
  • Notes de laboratoire
  • Notes personnelles
  • Documents techniques
  • Site web

Le cellulaire est interdit.

Aucun logiciel de communication ne doit être en fonction sur votre ordinateur portable.

1.5. Respect de la nomenclature​

Il est important de respecter la nomenclature demandée pour les éléments dans l'examen.

Si le dossier à créer est spécifié comme ceci : C:\PiLoTeS, il doit respecter la logique des majuscules et minuscules, même en Windows.

Si le nom n'a pas d'espace, il ne faut pas en ajouter.

2. Explication de l'examen​

L'examen est sur 100 points et il vaut pour 55% de la session.

La répartition des points sera la même que dans l'exemple ci-dessous.

Vous devez faire l'examen en 2 heures et 50 minutes.

Voici la répartition de l'examen avec sa pondération.

Partie 1 - Problème du BIOS - 4 points​

Il ne sera pas possible d'installer Windows, car il aura une configuration BIOS non fonctionnelle.

Les 3 possibilités que vous avez expérimentées sont :

  • Port SATA désactivé
  • Secure Boot activé
  • Port USB désactivé

Si vous ne trouvez pas la nature du problème, vous pouvez demander à l’enseignant de passer cette partie, mais aucun point ne sera alloué.

Partie 2 - Identification du matériel - 6 points​

Vous devez répondre à des questions sur la configuration en utilisant les outils diagnostiques.

  • BIOS
  • HWiNFO
  • WMIC (RamCheck)

Partie 3 - Installation de Windows - 40 points​

3.1 Installation de Windows

Le fichier ISO à utiliser pour l'installation sera « Win11_ExamenA23.iso ».

Il sera important de prendre le bon disque pour l'installation. Afin d'éviter de faire une erreur, il sera important de désactiver le port SATA dans le BIOS du disque inutilisé.

Il sera important de respecter les majuscules et les minuscules pour le nom de l'ordinateur et de l'utilisateur.

Il sera important de faire la conversion de Go à Mo en utilisant le facteur 1024.

Il sera important d'utiliser le port USB 3.

Pour vous pratiquer

Démarrez l’ordinateur avec votre clé USB.

Windows doit être installé sur une partition de 74 Go.

Le nom de l’ordinateur doit être Ordi#T@.

  • Le @ représente votre table de travail qui est inscrite sur la tablette.
  • Le # représente votre groupe.
  • Si votre table est la 9 et votre groupe 6, l’ordinateur doit s’appeler Ordi6T9.

Le nom de l’utilisateur doit être Etudiant1R1 et il ne doit pas avoir de mot de passe.

La disposition du clavier doit être Français (Canada).

3.2 Windows Update

Vous devez mettre en pause les mises à jour de Windows pendant 2 semaines.

Avertissement

L'enseignant devra valider cette étape.

Si Windows Update n'est pas en pause, il se mettra à jour pendant l'examen.

3.3 Installation des pilotes de bases

Installez tous les pilotes des périphériques de base de l’ordinateur.

Utilisez le pilote qui se retrouve sur votre clé sur la partition DRV de votre clé USB et le logiciel Dell Support Assist.

3.4 Gestion des logiciels

Vous devez installer des logiciels en fonction de l'architecture demandés.

Vous devez également désinstaller un logiciel.

Le programme d'installation des logiciels sera fourni par l'enseignant. Il ne faudra pas les télécharger.

Pour vous pratiquer

Installez les programmes qui se retrouvent dans le dossier C:\Installation.

3.5 Installation d'un 2e périphérique

Vous devez installer un 2e périphérique parmi ceux que vous avez déjà utilisés.

Les pilotes se retrouvent dans le fichier C:\Pilotes\pilotes.zip pour ceux qui ne sont plus disponibles sur internet.

  • Le pilote se retrouve dans le dossier qui correspond à l’identifiant du périphérique.

Pour ceux qui sont toujours disponibles en ligne, vous devez retrouver le fichier en ligne ou celui que vous avez conservé dans votre ordinateur.

Pour vous pratiquer

Installez les périphériques et installez les pilotes.

3.6 Configurations

Vous devez effectuer des configurations dans Windows.

Pour vous pratiquer

Regardez les configurations du laboratoire 1et 2. Ce sera des configurations que vous avez effectuées dans ces 2 laboratoires.

Partie 4 - Gestion des partitions en Windows - 10 points​

Vous devez faire des actions sur les partitions.

Je peux demander de choisir des options en fonction d'une question théorique du genre.

  • Je dois permettre d'avoir des fichiers de plus de 4 Go.

Vous devez créer des dossiers et des fichiers sur les partitions.

Pour vous pratiquer

1- Nommez la partition principale de Windows (lecteur C) pour « VOL_OS ».

2- Créez les partitions ci-dessous sur l’espace restant du disque utilisé pour l'installation de Windows.

NomSystème de fichiersTailleLettre
VOL_DATANTFS23 GoZ
VOL_TPexFAT49 GoY
VOL_UNIFAT3217 GoP

Partie 5 - Utilisation de Ubuntu Live - 20 points​

Vous devez utiliser Ubuntu Live pour faire des actions dans Linux.

Vous devez être en mesure de créer d'initialiser un disque et de créer des partitions avec GParted.

  • Je peux demander de choisir des options en fonction d'une question théorique du genre.
    • Je dois permettre d'avoir des fichiers de plus de 4 Go.

Vous devez être en mesure de monter des partitions.

Vous devez être en mesure de créer des dossiers et des fichiers sur les partitions.

Vous devez être en mesure de récupérer des fichiers sur la racine de la partition principale (/) et de les copier sur une autre partition.

Pour vous pratiquer

Réinitialisez le disque avant de créer les partitions.

Créez les 3 partitions ci-dessous avec GParted sur le disque.

Le disque doit avoir uniquement ces partitions.

Système de fichiersTailleÉtiquette/Label
NTFS42 GioVOL1
EXT452 GioVOL2
FAT3224 GioVOL3

Sur chacune des partitions, vous devez être en mesure de créer des dossiers et des fichiers textes.

Vous devez copier le fichier /etc/fstab à la racine de la partition VOL1 et VOL2.

Partie 6 - Utilisation de Windows PE - Invite de commandes - 20 points​

Vous devez être en mesure de trouver la lettre de lecteur pour une partition à l'aide de diskpart.

Conseil

Vous avez fait cette action dans le laboratoire 10 pour copier le fichier ISO pour l'installation de Windows.

Vous devez être en mesure de faire des actions en ligne de commandes.

  • Changer de lecteur
  • Changer de dossier de travail
  • Copier, déplacer et supprimer des fichiers
  • Créer des dossiers
  • Afficher le contenu des variables
  • Effectuer une redirection de sortie > pour une commande
  • Utiliser des métacaractères pour faire une action
Pour vous pratiquer
  • Identifiez la lettre de lecteur pour la partition VOL2 et VOL3 que vous avez créés à l'aide de diskpart.
  • Créez le dossier WPE sur la partition VOL3.
  • Copiez tous les fichiers du dossier x:\windows\system32 qui ont l'extension dll dans le dossier WPE de la partition VOL2.
  • Supprimez tous les fichiers du dossier WPE de la partition VOL2 qui contiennent un t dans leur nom.
  • Redirigez la commande dir du dossier x:\windows\system32 dans le fichier redirect.txt. Le fichier redirect.txt doit être sur la racine de la partition VOL3.
  • Affichez le contenu de la variable %WINDIR%.
