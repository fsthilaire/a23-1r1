Laboratoire de la semaine 5

Dans ce laboratoire, vous effectuerez l'analyse des différentes composantes avec les outils diagnostiques.

Ensuite, vous allez créer votre clé démarrable USB.

Finalement, vous effectuerez le démontage complet de l'ordinateur.

Démarrez l'ordinateur.

1. Identification des composantes​

Prenez des photos des différentes fenêtres pour l'identification des composantes.

Une fois l'ordinateur assemblé, il faut que ce soit identique.

Référez-vous à la section Procédures - Utilitaires pour savoir comment utiliser les outils.

1.1 Par le BIOS - Détection du matériel​

Identifiez les unités de stockage dans tous les ports SATA.

1.2 Par HWiNFO​

Insérez le disque SSD Samsung dans le tiroir.

Exécutez le logiciel c:\apps\hwinfo\hwinfo64.exe.

Notez les éléments suivants.

  • Nombre de barrettes de mémoires

  • La quantité totale de mémoire en Go / GB.

  • Le nombre de disques (HDD/SSD)

  • La taille de chacun des disques

  • Le nom du lecteur DVD

  • Les cartes réseau

1.3 Par WMIC​

Exécutez le logiciel c:\apps\wmic\ramcheck.bat.

Identifiez l'information de la mémoire pour chacun des emplacements DIMM.

2. Création de la clé démarrable USB​

2.1 Utilisation de Ventoy​

Vous devez faire cette étape avec les 2 clés USB si vous êtes en équipe.

Note

Référez-vous à Prodédures -> Utilitaires -> Ventoy pour avoir la procédure complète.

Insérez votre clé USB dans un port USB 3.0.

Exécutez le logiciel c:\apps\ventoy-1.0.95\Ventoy2Disk.exe.

Dans les options

  • Décochez Secure Boot Support.
  • Dans la section Partition Style, cochez GPT.

Appuyez sur le bouton Install.

2.2 Copie des fichiers ISO​

Vous devez faire cette étape avec les 2 clés USB si vous êtes en équipe.

Ouvrez l'Explorateur de fichiers.

Trouvez le lecteur qui se nomme Ventoy.

Copiez le fichier c:\apps\iso\WinPEx64_A23.iso vers le lecteur Ventoy.

Copiez le fichier c:\apps\iso\paragon17.iso vers le lecteur Ventoy.

La copie des 2 fichiers prendra environ 2 minutes pour une clé USB 3.0 qui est dans un port 3.0.

2.3 Test de la clé​

Éteignez l'ordinateur.

Retirez le disque SSD Samsung du tiroir.

Démarrez l'ordinateur et appuyez sur le touche F12 pour voir le menu de démarrage.

Dans la liste, sélectionnez la clé USB. Le nom varie en fonction du modèle de la clé.

Vous devriez voir le menu Ventoy avec le fichier WinPEx64__A23.iso.

Éteignez l'ordinateur et testez la 2e clé.

3. Consignes générales​

Classez vos pièces dans les tiroirs ou sur la tablette. L'organisation de votre espace est importante et ce point sera évalué lors de l'examen.

3.1 Libérer le boitier​

Il est important de déconnecter tous les items du boitier lorsqu'il faut travailler à l'intérieur du boitier.

  • Alimentation électrique
  • Clavier
  • Souris
  • Câble pour les écrans
  • Clé USB.

3.2 Gestion des câbles​

Il y a beaucoup de câbles dans le boitier.

Les câbles sont dans des guides dans le boitier.

Remarquez bien leur emplacement pour l'assemblage.

Par exemple, le câble du bouton d'alimentation est en dessous des câbles du connecteur principal du bloc d'alimentation. Le bloc d'alimentation s'enlève plus souvent que le bouton d'alimentation.

Une mauvaise gestion des câbles peut rendre le remplacement des pièces plus difficiles.

Un câble qui est libre peut également se déplacer dans le boitier et bloquer des ventilateurs.

Les câbles inutilisés doivent être retirés de l'ordinateur. Par exemple, s'il faut retirer un disque, il faut retirer son câble de données SATA.

Conseil

Vous pouvez prendre des photos de la gestion des câbles comme point de repère.

3.3 Câble de données SATA​

Avant de retirer les câbles de données SATA, il est important de prendre en note les éléments ci-dessous pour chacun des périphériques.

  • Le numéro de port SATA,
  • Le type de câble (Plat, Angle droit ou avec verrou)
  • La couleur

Lors de la réinstallation, il faut que la configuration des câbles de données SATA soit identique.

Conseil

Vous pouvez prendre des photos de la configuration comme point de repère.

3.4 Emplacement des périphériques​

Il est important de remettre les périphériques dans les mêmes emplacements.

Si le disque SSD est dans l'emplacement du bas pour les disques internes, il faut le remettre au même endroit.

Si le lecteur DVD est dans l'emplacement du haut dans l'emplacement des périphériques externes, il faut le remettre au même endroit.

Conseil

Vous pouvez prendre des photos de l'emplacement des périphériques comme point de repère.

3.5 Vérification de la configuration​

À la fin de l'assemblage, il faut vérifier la configuration avec les outils.

Pour être en mesure de tester le tiroir SATA, il faut mettre un disque à l'intérieur. S'il n'y a pas de disque, les outils ne seront pas en mesure de vérifier si tous les branchements sont effectués correctement.

3.6 Pièces à ne pas retirer​

Important

Voici les pièces qui ne seront pas retirées dans ce cours.

  • Le dissipateur de chaleur/ventilateur du processeur. Il sera possible de le faire avec du matériel non fonctionnel.

  • Le processeur ne sera jamais enlevé de la carte-mère. Il sera possible de le faire avec du matériel non fonctionnel.

  • La pile ne sera pas enlevée. Il sera possible de le faire avec du matériel non fonctionnel.

  • Le ventilateur du boitier ne sera pas retiré.

3.7 Processeur​

Le processeur devrait être la première pièce à être installée ou la dernière.

Le socket du processeur est très fragile. Une simple poussière peut empêcher le bon fonctionnement du processeur.

Il faut éviter de laisser le socket libre et sans protection.

Une nouvelle carte-mère a un dispositif de protection du socket pour imiter un processeur, car la carte-mère n'est pas livrée avec un processeur.

3.8 Dissipateur de chaleur/ventilateur​

Avant d'installer le dissipateur de chaleur/ventilateur, il faut nettoyer correctement le processeur et le dissipateur de chaleur/ventilateur pour retirer l'ancienne pâte thermique.

Ensuite, il faudrait rappliquer de la nouvelle pâte thermique.

La pâte thermique permet de combler les imperfections entre le processeur et le dissipateur de chaleur pour optimiser le contact entre les 2 pièces.

Il arrive parfois qu'il ne soit pas possible de réappliquer de la nouvelle pâte thermique. L'ordinateur va fonctionner correctement, mais le refroidissement du processeur sera moins efficace.

Voici un document explicatif comment l'appliquer. : https://www.intel.fr/content/www/fr/fr/gaming/resources/how-to-apply-thermal-paste.html

4. Démontage de l'ordinateur​

Retirez le disque SSD du tiroir.

4.1 Étudiant 1​

  • Retirez le capot.

  • Retirez le cadre avant.

  • Retirez le lecteur DVD. Pensez à déconnecter le câble de données et d'alimentation avant de le retirer. Il faut déconnecter le câble de données SATA de la carte-mère également.

  • Retirez le disque HDD. Il peut rester dans le support bleu. Il faut déconnecter le câble de données SATA de la carte-mère également.

  • Retirez la carte réseau du port PCI. Déposez la carte réseau dans le sac antistatique. Pensez à refermer le verrou pour que les protecteurs PCI restent en place.

  • Retirez le détecteur anti-intrusion.

  • Retirez le panneau d'entrée/sortie. Débranchez les 2 connecteurs de la carte-mère avant de l'enlever du boitier. Les connecteurs ont des boutons verrous.

4.2 Étudiant 2​

  • Retirez le tiroir de disque SATA. Pensez à déconnecter le câble de données et d'alimentation avant de le retirer. Il faut déconnecter le câble de données SATA de la carte-mère également.
  • Retirez la sonde thermique. Remarquez bien où elle est positionnée dans le boitier.
  • Déconnectez de la carte-mère le câble pour l'alimentation SATA.
  • Retirez le bloc d'alimentation du boitier et déconnectez ses 2 connecteurs sur la carte-mère.
  • Retirez la mémoire vive.
  • Retirez le bouton d'alimentation.
  • Débranchez le ventilateur du boitier de la carte-mère.
  • Retirez la carte-mère. Déposez la carte réseau dans le sac antistatique.

5. Assemblage​

N'oubliez pas de

  • Remettre les périphériques au même endroit
  • Réappliquer la même configuration pour les câbles de données SATA
  • Effectuez la gestion des câbles

5.1 Étudiant 1​

  • Installez la carte-mère.
  • Branchez le ventilateur du boitier sur la carte-mère.
  • Installez le bouton d'alimentation. Le câble doit être dans les guides.
  • Installez la mémoire vive.
  • Installez le bloc d'alimentation dans le boitier et connectez ses 2 connecteurs sur la carte-mère.
  • Installez la sonde thermique.
  • Connectez le câble d'alimentation SATA sur la carte-mère.
  • Installez le tiroir de disque SATA. Pensez à connecter le câble de données et d'alimentation. Il faut connecter le câble de données SATA sur la carte-mère également.

5.2 Étudiant 2​

  • Installez le panneau d'entrée/sortie. ATTENTION : les connecteurs ont un sens sur la carte-mère. Il est très facile de les briser. Validez avec l'enseignant en cas de besoin.

  • Installez le détecteur d'anti-intrusion.

  • Installez la carte réseau PCI.

  • Installez le lecteur DVD. Pensez à connecter le câble de données et d'alimentation. Il faut connecter le câble de données SATA sur la carte-mère également.

  • Installez le disque HDD. Pensez à connecter le câble de données et d'alimentation. Il faut connecter le câble de données SATA sur la carte-mère également.

  • Installez le cadre avant.

  • Installez le capot.

6. Vérification​

Vous devez valider vos branchements.

Si tout est identique à l'installation initiale, le laboratoire est terminé.

S'il y a des différences, il faut refaire les branchements correctement.

Demandez à l'enseignant de valider la gestion des câbles dans les guides.

6.1 BIOS​

Démarrez l'ordinateur et entrez le BIOS pour vérifier les SATA.

6.2 Windows PE​

Redémarrez l'ordinateur pour utiliser Windows PE.

Appuyez sur la touche F12 au démarrage pour avoir le menu.

Sélectionnez votre clé USB dans la liste.

Dans le menu Ventoy, sélectionnez WinPEx64_A23.iso.

Une fois que Windows PE est chargé en mémoire, effectuez les 2 commandes ci-dessous.

  • Exécutez HWiNFO en inscrivant hwinfo64.exe dans l'invite de commandes.

  • Exécutez WMIC en inscrivant ramcheck.bat dans l'invite de commandes.
