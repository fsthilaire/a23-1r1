Laboratoire de la semaine 2

Prenez la version de Windows de votre choix pour débuter. Le numéro du disque indique la version.

  • Windows 11 : Nombre impair (1R1-01, 1R1-03...)

  • Windows 10 : Nombre pair (1R1-02, 1R1-04...)

Utilisez l'utilisateur Semaine2. Le mot de passe est fghj.

1. Branchement de l'ordinateur​

Vous devez brancher correctement l'ordinateur de façon ergonomique.

L'ordinateur doit avoir 2 écrans.

L'écran LG doit être le principal (physique et dans la disposition de Windows).

L'écran DELL doit être à gauche de l'écran LG.

L'écran DELL doit utiliser le câble DVI.

Les écrans doivent être en mode étendu.

Pensez à mettre la bonne disposition dans Windows.

2. Disposition du clavier​

2.1 Créer un dossier​

Créez sur le bureau un dossier ZQMX4.

Il semble avoir un problème avec les lettres.

Quelle est la raison ? Identifiez la disposition du clavier de l'ordinateur pour comprendre le comportement.

Réponse

La disposition du clavier est ou Français ou Français (France).

Cette disposition utilise un clavier AZERTY, c'est à dire que les 5 premières lettres sont : A, Z, E, R, T, Y.

Les disponistions des claviers nord-américaines sont QWERTY.

Donc, c'est la disposition du clavier qui est configuré dans l'ordinateur qui détermine le caractère qui doit s'afficher. Le caractère imprimé sur le clavier n'a pas de réelle importance, mais il et préférable d'utiliser une clavier physique identique à la disposition qui est configuré dans l'ordinateur.

2.2 Modification de la disposition​

Veuillez avoir uniquement Français (Canada) et Anglais (État-Unis).

Il faut activer les raccourcis CRTL+MAJ et ALT Gauche + MAJ pour le changement de disposition.

Il faut afficher la barre de menu

3. Fichier masqué/caché et extension​

Affichez les extensions et les fichiers cachés dans l'Explorateur de fichier.

Ouvrez le dossier C:\lab2\e24.

Déterminez le nombre de fichiers que l'extension se termine par .txt.

Réponse

Il y a 1000 fichiers avec une extension .txt.

Il y en a 100 qui sont masqués et il y en a 100 que leur exteion est .rtf.

4. Démarrage de l'Explorateur de fichiers​

Il faut mettre Ce PC et non Accès rapide à l'ouverture de l'Explorateur de fichiers.

Créez le dossier C:\lab2\apps dans l'ordinateur.

Copiez tous les éléments de la clé USB du laboratoire dans le dossier C:\lab2\apps.

Épinglez le dossier apps dans les accès rapides de l'Explorateur de fichiers.

5. Installation de logiciels​

Les logiciels d'installation sont dans le dossier C:\lab2\apps.

5.1 Partie 1​

Navigateur Web

  • Google Chrome -> ChromeStandaloneSetup64.exe

Lecteur PDF

  • Sumatra PDF -> SumatraPDF-3.4.6-64-install.exe

5.2 Partie 2​

Vous devez sélectionner la bonne version du logiciel d'installation.

Rappel

Pour la version 64 bits, le programme contient généralement le suffixe x64.

Pour la version 32 bits, le programme contient le suffixe x86 ou x32.

Outil d'archive et de compression

  • 7-Zip -> 7z2201-x??.exe
  • Utilisez le fichier en fonction de l'architecture 64 bits

Éditeur de texte

  • NotePad++ -> npp.8.5.5.Installer.x??.exe
  • Utilisez le fichier en fonction de l'architecture 32 bits

6. Programme par défaut​

Il faut mettre Chrome comme navigateur par défaut.

Il faut mettre Chrome pour les fichiers .pdf.

Il faut mettre NotePad++ pour les fichiers .html.

Attention

Pour ce type de fichier, la procédure n'est pas dans les notes de cours.

Demandez à l'enseignant.

Il faut mettre 7-Zip pour la gestion des fichiers .zip par défaut pour tous les utilisateurs.

Attention

Vous devez le faire par le programme 7-Zip. Référez-vous à la section Procédures -> Utilitaires -> 7-Zip

Il ne faut pas le faire par les paramètres de Windows.

7. Désinstallation de logiciels​

Désinstallez le logiciel Opera.

8. Configuration du Bureau​

Il faut configurer les icônes système sur le Bureau. Il faut afficher uniquement Ordinateur et Corbeille.

S'il y a d'autres icônes système, il faut les retirer.

9. Barre de tâches​

Épinglez uniquement ces logiciels dans la barre de tâches.

  • Chrome
  • Explorateur de fichiers
  • Paramètres de Windows
  • 7-Zip File Manager

La barre de tâches doit être à gauche.

10. Alimentation​

Il faut désactiver les écrans après 30 minutes.

L'ordinateur ne doit jamais se mettre en veille.

11. Créer un fichier compressé - 7-Zip​

Compressez le contenu du dossier C:\lab2\apps.

Utilisez le format zip. Le fichier doit s'appeler MesApps.zip et il doit être sur le bureau.

12. Extraire un fichier compressé - 7-Zip​

Créez le dossier C:\extraction.

Il faut extraire le contenu du fichier C:\lab2\prog.zip dans le dossier C:\extraction

Le mot de passe est le nombre de fichiers qui sont cachés/masqués dans le dossier C:\Windows.

Le mot de passe est différent selon la version de Windows.

13. Extra​

13.1 3 écrans​

Effectuez une configuration à 3 écrans.

L'écran LG doit être au centre.

Il y a un écran DELL de chaque côté de l'écran.

Les écrans sont en mode étendu.

Pensez à bien disposer les écrans dans Windows.

13.2 Mode miroir​

Configurez les 2 écrans DELL pour être en mode duplication (miroir).

Dans la disposition de Windows, l'écran DELL (miroir) doit être au-dessus de l'écran LG

13.3 God Mode​

Les configurations ci-dessous ne sont pas dans les notes de cours. Veuillez faire des recherches pour trouver comment les réaliser.

Créez l'icône pour le God Mode de Windows.

L'information ne se retrouve pas dans les notes de cours. Effectuez une recherche sur le Web.

Indice

Il faut créer un dossier avec un code alphanumérique.

Les configurations ci-dessous se font par le God Mode

- Modifiez le comportement des boutons d’alimentations. Il faut que l’écran se ferme (Désactiver l’Affichage ou Éteindre l’écran) lorsque le bouton Marche/Arrêt est appuyé.

- Modifiez l’affichage de l’heure pour que le premier jour de la semaine soit mardi.

- Renommez cet ordinateur. Le nom doit être Init1R1. Ce doit être le nom et non la description.

13.4 Langue d'affichage​

Les configurations ci-dessous ne sont pas dans les notes de cours. Veuillez faire des recherches pour trouver comment les réaliser.

Changez la langue d'affichage de Windows par une langue de votre choix.

L'information ne se retrouve pas dans les notes de cours. Effectuez une recherche sur le Web.

14. Rangement​

Débranchez tous les câbles.

  • Tablette

    • Disques SSD
    • Adaptateur disque
    • Bracelet antistatique

  • Premier tiroir

    • Câble VGA et DVI
    • Adaptateur VGA et DVI
    • Attache-câbles

  • Deuxième tiroir

    • Clavier
    • Souris

  • Troisième tiroir

    • Câbles d'alimentation électrique
    • Bloc d'alimentation de l'écran LG
    • Câble réseau
