Laboratoire de la semaine 10

1. Problème de BIOS​

Insérez le disque Micron 256 Go SDD 2.5" qui a un numéro pair dans le tiroir SATA du haut.

Connectez les 2 clés USB à l'ordinateur si vous êtes en équipe.

Démarrez l'ordinateur avec Windows PE de votre clé USB.

Pourquoi qu'il n’est pas possible de démarrer sur la clé USB ?

Trouvez la raison dans le BIOS.

2. Copie du ISO d'installation​

Démarrez l'ordinateur avec Windows PE de l'une des 2 clés USB.

Pour connaitre la lettre des partitions, il faut utiliser DiskPart. Il faut faire la commande list volume pour obtenir l'information.

list volume

À partir de l'invite de commandes, copiez le fichier Win11_1R1.iso qui se retrouve dans la partition ISO du disque dur Seagate sur la partition Ventoy de votre clé USB.

Vous devez faire la copie du ISO pour les 2 clés USB.

Lorsque la copie est terminée, fermez l'ordinateur et retirez le Micron 256 Go SDD 2.5" dans le tiroir SATA du haut.

3. Installation de Windows​

Note

Référez-vous à la procédure qui se retrouve dans la section Procédures -> Installation de système d'exploitation -> Windows 11.

Vous devez installer Windows 11 sur le disque Seagate 500 Go HDD 3.5" qui se retrouve dans le boitier. Il est connecté dans le port SATA 0.

Démarrez l'ordinateur avec Win11_1R1.iso à partir de votre clé USB.

Windows doit être installé sur une partition de 100 Go. Effectuez la conversion en Mo correctement en utilisant le facteur 1024.

Le nom d'utilisateur doit être Etudiant et il ne doit pas avoir de mot de passe.

La disposition du clavier doit être Français (Canada).

Lorsque l'installation est terminée, éteignez l'ordinateur.

4. Installation d'un 2e écran​

Installez un 2e écran.

Démarrez l'ordinateur.

Est-ce que le 2e écran fonctionne ? Pourquoi ?

Réponse
Le 2e écran ne fonctionne pas, car le pilote de la carte graphique est un pilote de base. Ce pilote permet d'avoir un affichage minimal pour être en mesure de faire les premières configurations. Ce pilote n'est pas en mesure de gérer plusieurs écrans. Il faut donc installer le vrai pilote pour la carte graphique.

5. Installation des pilotes​

Important

Il n'y pas de procédure écrite pour cette section.

Il est important de prendre des notes lors de la démonstration et lors de votre expérimentation.

5.1. Carte réseau​

Allez sur le site de support de Dell à partir de votre portable et téléchargez le pilote de la carte réseau.

Conservez ce pilote dans votre ordinateur, il vous sera utile plusieurs fois.

5.2. Utilitaire Dell Support Assist​

Connectez le câble réseau pour avoir accès à Internet.

Allez sur le site du support de Dell.

Entrez le Service Tag de votre ordinateur pour accéder à sa page.

Dans la section Pilotes et téléchargements, téléchargez l'utilitaire du support de Dell et installez-le.

L'utilitaire va détecter les pilotes manquants et ensuite les installer automatiquement.

Vérifiez dans le Gestionnaire de périphériques que tous les périphériques sont correctement installés.

6. Configuration de Windows​

6.1. Nom de l'ordinateur​

Le nom de l'ordinateur doit être Gr#Tb@.

Le # représente votre numéro de groupe et le @ représente votre table de travail qui est inscrite sur la tablette.

Exemple, si vous êtes le groupe 5 et vous avez la table 2, l'ordinateur doit se nommer Gr5Tb2

Il est important de respecter les majuscules et minuscules.

6.2. Disposition des écrans​

L'écran LG doit être l'écran principal.

Vous devez indiquer l'écran principal dans la configuration Windows.

La souris doit se déplacer correctement d'un écran à l'autre.

6.3. Disposition du clavier​

Il faut avoir uniquement la disposition Français (Canada).

Il faut désactiver les raccourcis CTRL+MAJ et ALT Gauche + MAJ pour le changement de disposition.

6.4. Explorateur de fichiers​

Il faut afficher les extensions des fichiers.

Il faut afficher les fichiers masqués.

Il faut mettre Ce PC et non Accès rapide à l'ouverture de l'Explorateur de fichier.

6.5. Gestion de l'alimentation​

Il faut éteindre les écrans après 5 minutes.

Il faut ne faut jamais mettre l'ordinateur en veille.

6.6. Fichier​

Créez le fichier Étudiants.txt sur le bureau.

Inscrivez à l'intérieur votre prénom et nom.

7. Machine virtuelle​

Référez-vous aux procédures qui se retrouvent dans la section Procédures -> Utilitaires -> VirtualBox.

7.1. Installer VirtualBox​

Installez VirtualBox sur l'ordinateur.

Référez-vous à la procédure d'installation pour télécharger les 2 programmes.

7.2. Créer une machine virtuelle​

Vous devez créer une machine virtuelle selon les spécifications ci-dessous.

  • Nom : VM1
  • Type : Microsoft Windows
  • Version : Other Windows (64-bit)
  • Mémoire vive : 4096 Mo
  • Processeur : 1
  • Disque interne : 12 Gio

7.3. Ajouter un disque​

Ajoutez 2 autres disques internes à la machine virtuelle.

  • Disque interne 2 : 1 Gio
  • Disque interne 3 : 9 Gio

7.4. Assigner un iso​

Assignez le ISO de Windows PE au lecteur DVD.

Prenez celui de votre clé usb.

7.5. Démarrer la VM​

Démarrez la machine virtuelle.

Exécutez hwinfo64 pour faire la détection du matériel.

Vous devriez voir les 3 disques internes.

Regardez la mémoire vive. Est-ce comme d'habitude ?

7.6. Importer une VM​

Vous devez importer la machine virtuelle Windows XP.

Éteignez l'ordinateur et insérez le disque Micron 256 Go SSD 3.5" qui a un numéro pair dans le tiroir SATA.

Le fichier d'importation se retrouve sur la partition VM du disque Micron 256 Go SSD 3.5".

Exécutez le fichier Windows XP - Demo.ova.

7.7. Installation des Additions invité​

Installez l'outil des Additions invité dans Windows XP.

7.8. Modifier la mémoire vive​

Modifiez la mémoire vive de la machine virtuelle pour ajouter 8 Go (8192 MB).

Trouvez l'emplacement pour afficher la mémoire vive que Windows utilise.

Est-ce 8 Go ? Pourquoi ce n'est pas la même quantité ?

Réponse
L'architecture par défaut de Windows XP est de 32 bit. Pour un système 32 bit, il n'est pas possible d'avoir accès au-delà de 4 Go, même si l'ordinateur en possède plus.

7.9. Ajouter un disque interne​

Ajoutez un disque interne de 40 Gio.

Dans Windows, allez dans Gestion du disque. Exécutez la commande diskmgmt.msc.

1- Initialisez le disque en GPT.

Est-ce que l'option est disponible dans l'utilitaire Gestion du disque ?

Essayez avec diskpart.

select disk 1
clean
convert gpt

Pourquoi ce n'est pas possible de le faire ?

Réponse
Windows XP 32 bit n'est pas en mesure de travailler avec des partitions GPT.

2- Créez une partition en exFAT.

Initialisez le disque. Il sera en MBR.

Créez une partition qui prend l'entièreté du disque en exFAT.

Est-ce possible de le faire ? Pourquoi ?

Réponse
Le système de fichiers exFAT n'était pas disponible dans les premières versions de Windows XP.

8. Création d'une image avec Paragon​

Note

Référez-vous à la procédure qui se retrouve dans la section Procédures -> Utilitaires -> Paragon -> Créer une image.

Vous devez créer une image du disque Seagate 500 Go HDD 3.5". Cette image sera restaurée au prochain cours sur un SSD.

Fermez l'ordinateur.

Insérez le disque Micron 256 Go SSD 2.5" qui a le numéro impair dans le tiroir SATA du bas.

Démarrez l'ordinateur avec votre clé USB et sélectionnez paragon17.iso.

Vous devez créer un dossier sur la partition de votre groupe. Le dossier doit avoir comme nom ce modèle Gr#Tb@. Le # représente votre numéro de groupe et le @ représente votre table de travail qui est inscrite sur la tablette.

Exemple : Si la lettre D est assignée à la partition Img_PA et que vous être le groupe 5 et votre table est la 2, vous devez créer votre image dans le dossier D:\Gr5Tb2.

Important

Le disque Micron 256 Go SSD 2.5" IMPAIR sera partagé entre tous les groupes.

Il est important de ne pas effacer son contenu.

9. Réinitialisation du disque Seagate 500 Go HDD 3.5"​

Retirez le disque Micron 256 Go SSD 2.5" IMPAIR dans le tiroir SATA du bas.

Démarrez avec Windows PE à partir de votre clé USB.

Lorsque vous êtes dans Windows PE, retirez votre clé USB pour éviter de la réinitialiser par erreur.

Il faut réinitialiser le disque Seagate 500 Go HDD 3.5" à partir de DiskPart.

Tapez la commande diskpart pour démarrer le logiciel.

La commande ci-dessous permet d'afficher les unités de stockage.

list disk

Identifiez le numéro de disque correspondant au disque de 500 Go. Il sera en notation 1024, donc 465 Gio.

La commande ci-dessous permet de sélectionner le disque dur. Il faut remplacer le ### par le numéro du disque dur identifié à l'étape précédente.

select disk ###

Exemple

select disk 0

Ensuite, les commandes ci-dessous permettent la réinitialisation.

clean
convert gpt

Redémarrez l'ordinateur. Vous devez avoir la mention No bootable device.
