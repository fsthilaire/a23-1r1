Laboratoire de la semaine 1

1. Branchement de l'ordinateur​

Branchez l'ordinateur pour qu'il démarre.

Prenez les câbles nécessaires à son fonctionnement.

Tous les câbles se retrouvent dans le premier tiroir de votre bureau.

2. Ergonomie​

Vous devez installer le poste de travail de façon ergonomique.

3. Installation du disque SSD​

Partie 1​

Prenez le disque SSD avec un identifiant impair (1R1-01, 1R1-03,..., 1R1-25,...).

Insérez le disque dans l'adaptateur 2.5 à 3.5.

Insérez l'adaptateur dans le tiroir.

Démarrez l'ordinateur.

Utilisez l'utilisateur Semaine1. Le mot de passe est Labo1q.

Identifiez la version de Windows, son édition et son numéro de build.

Réponse
Version : Windows 11 22H2
Build : 22621.2134
Édition : Profesionnel

Partie 2​

Prenez le disque SSD avec un identifiant pair (1R1-02, 1R1-04,..., 1R1-26,...).

Insérez le disque dans l'adaptateur 2.5 à 3.5.

Insérez l'adaptateur dans le tiroir.

Démarrez l'ordinateur.

Utilisez l'utilisateur Semaine1. Le mot de passe est Labo1q.

Identifiez la version de Windows, son édition et son numéro de build.

Réponse
Version : Windows 10 22H2
Build : 19045.3324
Édition : Profesionnel

Partie 3​

Déterminez celui de votre portable.

Pour faire le reste des manipulations, prenez le disque dur correspondant au Windows de votre portable.

Si votre portable utilise Windows 11, prenez le disque impair pour Windows 11.

Si votre portable utilise Windows 10, prenez le disque pair pour Windows 10.

4. 2e écran​

Il faut installer un 2e écran.

L'écran Dell doit être l'écran principal et il doit être à gauche.

L'écran LG doit être à droite.

Effectuez les configurations dans Windows pour que le positionnement des écrans dans Windows représente la disposition physique.

Vous devez indiquer l'écran principal dans la configuration Windows.

La souris doit se déplacer correctement d'un écran à l'autre.

5. Résolution​

Déterminez la résolution de chacun des écrans.

6. Disposition du clavier​

Identifiez les dispositions des claviers de disponibles dans l'ordinateur.

Veuillez avoir uniquement Français (Canada) et Anglais (Canada).

Il faut désactiver les raccourcis CRTL+MAJ et ALT Gauche + MAJ pour le changement de disposition.

7. Affichage des extensions​

Sur le Bureau, vous avez seulement des fichiers nommés Laboratoire1.

Vous devez supprimer celui qui a l'extension .rtf.

Pour y arrivez, vous devez afficher les extensions des fichiers dans l'Explorateur de fichiers.

8. Copie de fichiers​

Créez un dossier sur le bureau nommé Logiciels.

Copiez tous les éléments de la clé USB du laboratoire dans le dossier Logiciels.

9. Installation de logiciels​

Les logiciels d'installation sont sur la clé USB du laboratoire.

Navigateur Web

  • Google Chrome - ChromeStandaloneSetup64.exe
  • Firefox Setup 116.0.2.exe

Lecteur PDF

  • Sumatra PDF - SumatraPDF-3.4.6-64-install.exe

10. Installation du VPN​

Partie 1​

Demandez le câble réseau à l'enseignant.

Branchez le câble réseau de la prise à l'ordinateur.

Veuillez installer le VPN du cégep sur l'ordinateur.

Testez vos identifiants pour vérifier que vous êtes bien connectés.

Partie 2​

Si ce n'est pas déjà fait, veuillez installer le VPN.

Vous devez obligatoirement le faire avant le prochain cours de Initiation à la programmation 420-1N1-DM.

11. Autre Windows​

S'il vous reste du temps, refaites les exercices 4 à 9 avec l'autre version de Windows.
