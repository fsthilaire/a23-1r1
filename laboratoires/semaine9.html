Laboratoire de la semaine 9

1. Réinitialisation du mot de passe​

Le BIOS est protégé par un mot de passe pour empêcher les modifications des paramètres.

Vous devez effacer le mot de passe du BIOS.

Consultez le manuel de l'utilisateur pour avoir la procédure. L'information se retrouve sous la section Mot de passe système et de configuration ou System and Setup Password.

Indice : C'est une manipulation à effectuer sur la carte-mère.

2. Configuration du BIOS​

  1. Il faut enlever la demande du mot de passe lorsqu'on démarre sur une clé USB.

    • General -> UEFI Boot Path Security : Never

      Cette option permet d'indiquer si le mot de passe système est nécessaire ou non lors du démarrage.

  1. Il faut enlever le mode de démarrage PXE.

    • System Configuration -> Integrated NIC : Enabled

      Cette option permet de configurer la carte réseau interne. Le mode PXE consiste à rechercher un serveur de démarrage pour charger le système d'exploitation à partir du réseau,

  1. Il faut modifier le mode de communication pour les ports SATA.

    • System Configuration -> SATA Operation : AHCI

      Cette option permet de configurer le contrôleur SATA pour déterminer le mode de communication.

  2. Il faut désactiver le Secure Boot.

    • Secure Boot -> Secure Boot Enable : Disabled

      Cette option consiste à vérifier si le périphérique de démarrage est sécurisé dans le cas qu'elle soit activée.

  3. Il faut mettre la bonne date et heure pour l'ordinateur.

    • General -> Date/Time : L'heure est en notation 12 heures AM/PM

Appuyez sur Apply pour enregistrer les modifications.

3. Insertion des disques​

Dans le tiroir SATA du haut, il faut insérez le disque Samsung 250 Go SSD 2.5".

Dans le tiroir SATA du bas, il faut insérez le disque Micron 256 Go SSD 2.5".

4. Restauration d'une image​

Note

Référez-vous à la procédure qui se retrouve dans la section Procédures -> Utilitaires -> Paragon -> Restaurer une image.

À partir d'une image Paragon, il faut restaurer une installation de Windows sur le disque de Samsung 250 Go SSD 2.5".

L'image se retrouve sur le disque Micron 256 Go SSD 2.5".

  1. Démarrez l'ordinateur avec votre clé USB.

  2. Dans le menu Ventoy, sélectionnez Paragon17.iso.

  3. L'image se retrouve dans le dossier Win11_Lab9 sur la partition IMG_PA.

  4. Lorsque c'est terminé, fermez l'ordinateur.

  5. Retirez le disque Micron 256 Go SSD 2.5" qui se retrouve dans le tiroir du bas.

  6. Retirez la clé USB.

  7. Redémarrez l'ordinateur et appuyez sur F12 pour démarrer sur le Samsung SSD 860 EVO 250GB.

  8. L'ordinateur démarrera dans Windows.

5. Invite de commandes​

Faites les actions ci-dessous avec l'invite de commandes.

Utilisez la notation absolue pour tous les numéros.

  1. Créez le dossier c:\lab9\dll.

    Réponse
    md c:\lab9\dll

  2. Créez le dossier c:\lab9\iso.

    Réponse
    md c:\lab9\iso

  1. Créez le dossier c:\lab9\apps.

    Réponse
    md c:\lab9\apps

  2. Copiez tous les fichiers .dll du dossier c:\windows\system32 dans le dossier c:\lab9\dll.

    Réponse
    copy c:\windows\system32\*.dll c:\lab9\dll\*.dll

  1. Copiez le fichier WinPEx64_A23.iso qui se retrouve sur votre clé USB dans le dossier c:\lab9\iso.

    Réponse
    copy d:\WinPEx64_A23.iso c:\lab9\iso\WinPex64_A23.iso Le lecteur d: peut être différent dans votre cas.

  1. Déplacez le fichier c:\apps\hwinfo\hwinfo64.exe dans le dossier c:\lab9\apps

    Réponse
    move c:\apps\hwinfo\hwinfo64.exe c:\lab9\apps\hwinfo64.exe

  1. Changez le dossier de travail pour c:\lab9\dll

    Réponse
    cd c:\lab9\dll

  1. Affichez tous les fichiers que leur nom débute par du.

    Réponse
    dir du*.*

  1. Renommez tous les fichiers que leur nom débute par z. Le z doit devenir 1r1_z.

    Réponse
    ren z*.* 1r1_z*.*

  1. Copiez tous les fichiers .msc du dossier c:\windows\syswow64 dans le dossier c:\lab9\dll.

    Réponse
    copy c:\windows\syswow64\*.msc c:\lab9\dll\*.msc

  1. Supprimez tous les fichiers .dll du dossier c:\lab9\dll.

    Réponse
    del c:\lab9\dll\*.dll

  1. Renommez le dossier c:\lab9\apps par prog.

    Réponse
    ren c:\lab9\apps prog

  2. Changez le dossier de travail pour c:\lab9.

    Réponse
    cd c:\lab9

  1. Supprimez le dossier c:\lab9\dll.

    Réponse
    rd c:\lab9\dll /s

  2. Affichez la variable %username% et %windir% (2 commandes).

    Réponse
    echo %username%
    echo %windir%

  3. Redirigez l'affichage des commandes précédentes dans le fichier c:\lab9\var.txt. Si le fichier existe pour la première variable, il doit être écrasé. Pour la 2e variable, la valeur doit être ajoutée au fichier existant.

    Réponse
    echo %username% > c:\lab9\var.txt
    echo %windir% >> c:\lab9\var.txt

7. Temps libre pour refaire les exercices​

Vous pouvez refaire les exercices de la semaine 5 pour la gestion des disques et ceux de la semaine 6, 7 et 9 pour l'invites de commande.

8. Effacer le disque​

Fermez l'ordinateur.

Le tiroir SATA du haut doit toujours avoir le disque Samsung 250 Go SSD 2.5" .

Le tiroir SATA du bas doit être vide.

Démarrez sur votre clé USB.

Dans Ventoy, sélectionnez WinPEx64_A23.iso.

Lorsque Windows PE est complètement chargé, retirez la clé USB. Windows PE fonctionne toujours, car tous les fichiers nécessaires à son fonctionnement sont dans la mémoire vive.

Dans l'invite de commandes, inscrivez les commandes ci-dessous. DiskPart est l'utilitaire en invite de commandes pour faire la gestion des disques.

diskpart

Dans DiskPart, inscrivez les commandes ci-dessous pour effacer complètement le disque.

select disk 0
clean
exit

Fermez l'ordinateur.

Retirez le disque Samsung 250 Go SSD 2.5" du tiroir SATA.

9. BIOS​

Il faut remettre le BIOS en désordre pour le prochain groupe.

  1. Il faut mettre la demande du mot de passe lorsqu'on démarre sur une clé USB.

    • General -> UEFI Boot Path Security : Always

  2. Il faut mettre le mode de démarrage PXE.

    • System Configuration -> Integrated NIC : Enabled w/PXE

  3. Il faut modifier le mode de communication pour les ports SATA.

    • System Configuration -> SATA Operation : RAID On

  4. Il faut activer le Secure Boot.

    • Secure Boot -> Secure Boot Enable : Enabled

  5. Il faut mettre un mot de passe administrateur dans le BIOS.

    • Security -> Admin Password : Mettre un mot de passe de votre choix.

  6. Il faut mettre une mauvaise date et heure pour l'ordinateur.

  • General -> Date/Time : L'heure est en notation 12 heures AM/PM

Appuyez sur Apply pour enregistrer les modifications.
