Laboratoire de la semaine 6

Dans ce laboratoire, vous devrez effectuer plusieurs configurations.

Pour chacune des configurations, il faudra vérifier la configuration avec les outils pour s'assurer qu'elle répond aux exigences.

Il faudra également valider la configuration de la mémoire vive pour s'assurer qu'elle soit correctement installée selon les règles du multi-canal.

1. Préparation des ordinateurs​

Vous avez 2 ordinateurs devant vous.

Pour chacun des ordinateurs, vous devez installer les pièces ci-dessous.

L'étudiant 1 fait le travail préparatoire sur l'ordinateur Optiplex 9020.

L'étudiant 2 fait le travail préparatoire sur l'ordinateur Optiplex 7010.

Vous devez faire un ordinateur à la fois. Ne pas oublier le bracelet antistatique.

N'oubliez pas de faire une gestion adéquate des câbles.

Effectuez l'installation ou les branchements dans cet ordre pour faciliter la gestion des câbles.

  • Bouton d'alimentation
  • Bloc d'alimentation (Ils sont différents en fonction du modèle)
  • Câble d'alimentation SATA (Uniquement pour le 9020)
  • Détecteur anti-intrusion.
  • Le ventilateur du boitier de la carte-mère
  • La sonde thermique
    • La sonde est connectée sur la carte, mais elle n'est pas installée sur le boitier.
  • Cadre avant
    • Attention de prendre celui correspondant au bon modèle d'ordinateur.
    • Comparez la carte-mère en fonction de l'image du manuel technique
      • Optiplex 9020 : Page 28 Français, Page 29 Anglais
      • Optiplex 7010 : Page 35 Français et Anglais

Lorsque toutes ces composantes sont correctement installées, il s'agit de l'état initial pour la configuration.

Avant de procéder à l'étape suivante, veuillez faire vérifier votre installation par l'enseignant.

2. Premier test​

Démarrez l'ordinateur Optiplex 7010.

Vous allez avoir un signal sonore.

Il s'agit d'un code d'erreur.

Identifiez dans le manuel technique la source du code d'erreur.

La liste des codes se retrouve dans la section 5 Dépannage de l'ordinateur.

  • La sous-section Diagnostics par le voyant d'alimentation est le code produit bouton d'alimentation lorsqu'il est orange.
  • La sous-section Codes de bips est pour le signal sonore.

3. Configuration​

Chacune des configurations est pour un client ou un utilisateur de votre entreprise.

Il est important de respecter les demandes du client/utilisateur.

Vous avez 10 configurations à effectuer pour 10 clients. Section 3.1 à 3.10.

Les éléments de configuration de l'ordinateur concernent à l'installation des pièces ci-dessous.

  • Mémoire vive
  • Unité de stockage
  • Périphérique PCI
    • Carte réseau
    • Carte graphique
  • Périphérique du cadre avant
    • Lecteur DVD
    • Tiroir SATA

Pour chacune des configurations, il est important de respecter les éléments ci-dessous.

  • Bon ordinateur
  • Couleur du câble SATA (si spécifié)
  • Port SATA
  • Emplacement des pièces dans le bon endroit
    • Emplacement pour les unités de stockage dans le boitier
    • Emplacement pour des périphériques du cadre avant.
  • L'utilisation de la bonne pièce
    • Type
    • Capacité
    • Taille
    • Modèle
    • Marque
  • Retirer les câbles SATA non utilisés
  • Pour les emplacements inutilisés pour les périphériques du cadre avant et PCI, il est important de remettre les protecteurs en place pour ne pas avoir accès à l'intérieur du boitier.

Pour la configuration de la mémoire vive, elle doit respecter les principes du multi-canal.

  • Même marque
  • Même modèle
  • Même capacité

Il est possible d'avoir un nombre impair de mémoires vives.

Ne pas tenir compte de la vitesse pour déterminer la performance. L'utilisation du multi-canal détermine la meilleure performance pour ce laboratoire.

Il doit toujours avoir une mémoire vive dans la DIMM 1. La DIMM 4 est seulement utilisée si les 4 emplacements sont utilisés.

remarque

Pour ce laboratoire, il y aura des configurations non valides en fonction du manuel technique pour la mémoire vive, afin de pratiquer plusieurs cas.

Pour chacune des configurations, il faut vérifier la conformité avec les 3 outils.

  • HWiNFO (hwinfo64.exe)
  • WMIC (ramcheck.bat)
  • BIOS (System Configurations -> Drives)

Lorsque votre vérification est terminée, demandez à l'enseignant de vérifier.

4. Détection de la version USB​

Utilisez l'ordinateur Optiplex 9020.

Pour chacun des périphériques USB, déterminez leur version.

Pour faire les tests, il faut utiliser le port USB le plus rapide.

Utilisez l'utilitaire USBDeview à partir de Windows PE.

  • Clavier
  • Carte réseau WIFI USB
  • Clé USB laboratoire
  • Webcam
Réponse

Clavier : 2.00

Carte réseu WIFI USB : 2.00

Clé USB Laboratoire : 2.00

Webcam : 1.10

5. Démontage​

Veuillez remettre les ordinateurs à leur état initial.

5.1 Optiplex 7010​

Retirez le cadre avant.

Retirez le bloc d'alimentation.

Retirez le bouton d'alimentation.

Retirez le détecteur anti-intrusion.

Débranchez le ventilateur du boitier de la carte-mère.

5.2 Optiplex 9020​

Retirez le cadre avant.

Retirez tous les câbles SATA.

Retirez les disques internes.

Retirez le tiroir SATA.

Retirez les câbles alimentions SATA

Retirez la mémoire vive.

Retirez les cartes PCI.

Retirez le bloc d'alimentation.

Retirez le bouton d'alimentation.

Retirez le détecteur anti-intrusion.

Débranchez le ventilateur du boitier de la carte-mère.
